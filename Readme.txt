/// Backend:
Node
Express
SQL SERVER


Paso 1. Al descargar el proyectos deben ejecutar el comando npm install en cada modulo y en la app.
Paso 2. Para iniciar la base de datos deben ir al modulo prestamos-db y ejecutar el comando npm run setup y se crearan las tablas con sus relaciones.
Paso 3. Ejecuta el modulo prestamos-api con el comando npm start.
Paso 4. Ejecutar la app prestamos-app con el comando npm start o ng serve --open 

Nota: La configuración de la base de datos del modulo prestamos-api y prestamos-db están en el archivo util.js
